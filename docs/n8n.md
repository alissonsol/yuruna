# n8n

Installs [n8n](https://n8n.io/) workflow automation.

| Guest | Command |
|---|---|
| **Amazon Linux** | `/automation/fetch-and-execute.sh guest/amazon.linux/amazon.linux.n8n.sh` |
| **Ubuntu Desktop** | `/automation/fetch-and-execute.sh guest/ubuntu.desktop/ubuntu.desktop.n8n.sh` |
| **Ubuntu Server** | `/automation/fetch-and-execute.sh guest/ubuntu.server/ubuntu.server.n8n.sh` |

Verify + run:

```bash
n8n --version
n8n start     # open http://localhost:5678
```

Full docs: [n8n.io/docs](https://docs.n8n.io/).
