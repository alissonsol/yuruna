# Virtual Development Environment (VDE)

Your first task is to establish a **Virtual Development Environment (VDE)**. The VDE gives you a consistent, reproducible workspace that mirrors our target platforms.

## 1. Host Setup

Pick the environment that matches your host machine and follow the linked instructions to create your first VM.

- [macOS UTM host](host.macos.utm/README.md)
- [Windows Hyper-V host](host.windows.hyper-v/README.md)

## 2. Guest Setup

Each host guide links to a guest-specific guide for creating the VM. After creating the VM, follow the guest instructions for post-VDE setup:

- [Amazon Linux guest](guest.amazon.linux/README.md)
- [Ubuntu Desktop guest](guest.ubuntu.desktop/README.md)
- [Windows 11 guest](guest.windows.11/README.md)
